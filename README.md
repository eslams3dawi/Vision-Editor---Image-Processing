# 🔬 Vision Editor – Advanced Image Enhancement & Analysis

A complete Image Processing & Computer Vision desktop application developed using Python and OpenCV as part of Image Processing Course Practical Exam.

The project focuses on image enhancement, filtering, edge detection, zooming techniques, histogram analysis, and Computer Vision accuracy improvement using preprocessing pipelines.

# 🧠 Project Idea

The project demonstrates the importance of image preprocessing in Computer Vision systems using the principle:

> “Garbage In = Garbage Out”

We simulated low-quality images by adding:
- Darkness
- Gaussian Noise

Then we applied an enhancement pipeline including:
- Bilateral Filtering
- Gamma Correction
- CLAHE
- Unsharp Masking

Finally, we compared face detection accuracy before and after enhancement.

# 📈 CV Accuracy Challenge

The application compares:
1. Clean Image
2. Degraded Image
3. Enhanced Image

Metrics:
- Number of Faces Detected
- Accuracy Percentage
- Inference Time

This demonstrates how preprocessing can improve Computer Vision performance.
